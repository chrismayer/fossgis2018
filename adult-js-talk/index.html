Best Practices

Node.js - JavaScript auf dem Server

  • Serverseitige JavaScript Plattform
  • Verarbeitung von großen Datenmengen durch nicht blockierendes IO-System
  • Event-basiert
  • Ideal für Webanwendungen
  • Open Source (MIT)
  • Vortrag "Serverseitiges JavaScript und GIS Fossgis 2015"

Node.js - für die Cliententwicklung

  • Meist im Zusammenspiel mit npm
  • Oft im Bereich der Frontend-Entwicklung
    • Webserver
    • Dependency-Management
    • Build-Tools
    • Unit Testing
